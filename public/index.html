API Sampler: Temperature microservice

User stories:

  1. I can pass a temperature in Celcius as a parameter, and it will give the equivalent temperature
    in Fahrenheit
  2. I can pass a temperature in Fahrenheit as a parameter, and it will give the equivalent temperature
    in Celcius
  3. I can pass a temperature without a label as a parameter, and it will give the equivalent temperature
    in Celcius (default behavior)

Example usage:

https://mighty-badlands-45305.herokuapp.com/28C
https://mighty-badlands-45305.herokuapp.com/-20F
https://mighty-badlands-45305.herokuapp.com/34

Example output:

{ "temperature": 82.4, "degrees": "Fahrenheit" }
{ "temperature": -28.88888, "degrees": "Celcius" }
{ "temperature": 93.2, "degrees": "Fahrenheit" }
